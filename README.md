# Md Nazmul Islam - Personal Website

This repository contains my personal website, portfolio, and blog.

## Live Site

- `https://mdnazmulislam0087.github.io/`

## What I Use

- Frontend: HTML, CSS, JavaScript
- Backend services: Supabase (auth, database, storage)
- Hosting: GitHub Pages

## Main Pages

- `index.html` - Home
- `about.html` - About
- `research.html` - Research
- `blog.html` - Blog list
- `post.html` - Single post
- `contact.html` - Contact
- `admin-login.html` - Admin login
- `admin-dashboard.html` - Content and post manager

## Content Workflow

- I update website text and blog posts from the Admin Dashboard.
- Published posts are stored in Supabase and appear on the live site automatically.
- Code/design updates are done in this repository and deployed through GitHub.

## Local Run

```powershell
python -m http.server 8080
```

Open `http://localhost:8080`.

## Deploy

1. Commit and push to GitHub.
2. GitHub Actions deploys to Pages.
3. Visit the live URL.

## Note

This is a personal website project and evolves continuously.
